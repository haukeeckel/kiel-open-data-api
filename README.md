# kiel-dashboard-api

Backend/API for a small city dashboard prototype based on Kiel Open Data.

## Requirements

- Node.js (LTS recommended)
- pnpm

## Setup

```bash
pnpm install
```

## Development

Note: pnpm dev will be added once the server exists.

## Quality checks

Lint:

```bash
pnpm lint
```

Typecheck:

```bash
pnpm typecheck
```

Format check:

```bash
pnpm format:check
```

Auto-format:

```bash
pnpm format
```
